Méjico-API

Esta API devuelve diferentes platillos que podrías encontrar en un restaurante mexicano.

Endpoints

Categorías

Con este endpoint obtendrás las categorías disponibles:

  • {{url}}/categorias?limite=20&desde=0

    • Parámetros:

  • limite: Limita el número de resultados (opcional).
  • desde: Para indicar a partir de cual categoría te interesa obtener resultados (opcional).

    Productos por Categoría

    Con este endpoint obtendrás los productos de una categoría en específico:

  • {{url}}/buscar/productos/nombre_categoria?limite=0&desde=0

    • Parámetros:

  • nombre_categoria: Debes poner el nombre de la categoría (entradas, plato fuerte, postres o bebidas) de los productos que te interesan.
  • limite: Limita el número de resultados (opcional).
  • desde: Para indicar a partir de cual producto te interesa obtener resultados (opcional).
    •                         
    La respuesta es similar a la búsqueda de productos.

    Productos por nombre

    Con este endpoint obtendrás los productos que coincidan con el nombre que indiques.

  • {{url}}/buscar/productos/nombre/nombre_producto

    • Parámetros:

  • nombre_producto: Debes poner el nombre del producto que quieres buscar.
    •                         
    La respuesta es similar a la búsqueda de productos.
